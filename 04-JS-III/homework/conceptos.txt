* Arrays

Un array es una lista de elementos; los elementos pueden ser de distintos tipos, como string, número, booleano, array, función, etc...
El array se declara así: var nombre_array = []; este es un array vacío. También se puede declarar con elementos: var nombre_array = [1, 2, 3, false, ['a', 'b', 'c']].
El array tiene una propiedad length que devuelve el total de elementos; para acceder al primer elemento: nombre_array[0], y al último: nombre_array[nombre_array.length - 1].
El array tiene funciones para manipular sus elementos: push, pop, shift, unshift, etc...
